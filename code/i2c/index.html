2-wire Interface

I haven't actually implemented any I2C devices on the digital compass but it looked promising for some type of devices, particularly the serial non-voltile RAMS and EEPROMs. The information here is for my own referencence. The I2C interface is a simple 2-wire bus that allows specialized IC's like a/d converters, digital potenciometers, serial eeproms, etc. to be connected to computers and microcontrollers.
