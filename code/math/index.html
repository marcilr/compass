Math

The compass requires many different math function to write graphics to the display and convert sensor data into bearing information. The math functions fall into the three categories of signed addition/subtraction, multiplication, and trigonometric functions. Originally I thought I would be able to find 8051/52 math libraries on the net. I quickly found that there are few floating point libraries and scattered bits of code but none of the fixed point math routines I needed excluding commercial libraries and compilers. So I did what any good programmer does - I wrote my own.

  • 8-bit SIN and COS functions
  • CORDIC routines for 16-bit SIN, COS, ATAN.
  • Trig Function using Taylor Series (old)
  • 16.16 Fixed point routines
