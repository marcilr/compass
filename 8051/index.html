8051 Programming

Interrupt Vectors

When the original 8051 and 8031 were introduced, only 5 interrupts were provided.

Interrupt Number   Description      Interrupt Vector Address
------------------------------------------------------------
0                  EXTERNAL 0        0003h
1                  TIMER/COUNTER 0   000Bh
2                  EXTERNAL 1        0013h
3                  TIMER/COUNTER 1   001Bh
4                  SERIAL PORT       0023h
